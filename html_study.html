문서 구조

  • <html> : HTML 문서 전체를 감싸는 최상위 태그입니다.
  • <head> : 문서의 메타데이터를 포함하는 영역입니다.
  • <body> : 실제로 브라우저에 표시되는 내용을 담는 영역입니다.
  • <meta> : 문서의 문자 인코딩, 뷰포트 등 메타 정보를 제공합니다.

텍스트와 줄바꿈

  • <p> : 단락을 정의할 때 사용합니다.
  • <br> : 줄바꿈을 삽입할 때 사용합니다.
  • <strong> : 중요한 텍스트를 굵게 표시합니다.
  • <em> : 강조할 텍스트를 이탤릭체로 표시합니다.

이미지

  • <img> : 이미지를 삽입할 때 사용하며, src로 경로를 지정합니다.

표 (Table)

  • <table> : 표의 전체 구조를 정의합니다.
  • <tr> : 표의 한 행(row)을 정의합니다.
  • <td> : 표의 셀(열, 데이터)을 정의합니다.
  • <th> : 표의 제목 셀(굵게, 가운데 정렬)을 정의합니다.
  • colspan, rowspan : 셀 병합을 위한 속성입니다.

폼 (Form)

  • <form> : 사용자 입력을 서버로 전송하기 위한 영역입니다.
  • <input type="text"> : 텍스트를 입력할 수 있는 칸을 만듭니다.
  • <select> : 드롭다운 목록을 만듭니다.
  • <input type="radio"> : 여러 항목 중 하나만 선택할 수 있는 버튼입니다.
  • <input type="checkbox"> : 다중 선택 가능한 체크박스입니다.
  • <input type="file"> : 파일을 업로드하는 입력 필드입니다.
  • <button> : 클릭 가능한 버튼을 생성합니다.
  • <input type="hidden"> : 화면에는 보이지 않지만 서버로 전송되는 값을 저장합니다.
  • <label> : 폼 요소에 설명을 붙이는 데 사용합니다.
  • method : 데이터 전송 방식을 설정하는 속성입니다 (GET 또는 POST).
