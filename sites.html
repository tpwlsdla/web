추천사이트

웹 개발 학습 및 표준 정보 확인에 유용한 사이트들을 소개합니다.

ECMAScript 표준 홈 - 에카 인터내셔널

ECMAScript는 JavaScript의 공식 표준 명세입니다. 이 사이트에서는 ECMAScript의 최신 표준 문서와 기술 보고서를 확인하여, JavaScript 언어의 변화와 명세를 깊이 있게 이해할 수 있습니다.

사이트 링크: ecma-international.org

MDN (Mozilla Developer Network)

MDN Web Docs는 HTML, CSS, JavaScript 및 다양한 웹 API에 대한 광범위하고 신뢰할 수 있는 문서를 제공하는 필수 웹 개발 리소스입니다. 모든 수준의 개발자에게 유용하며, 특히 한국어 문서가 잘 되어 있어 학습에 큰 도움이 됩니다.

사이트 링크: developer.mozilla.org/ko/

W3Schools 온라인 웹 자습서

W3Schools는 웹 개발의 다양한 영역(HTML, CSS, JavaScript, SQL 등)에 대한 실용적인 자습서와 예제를 제공합니다. 직접 코드를 실행해 볼 수 있는 'Try it Yourself' 기능을 통해 학습한 내용을 바로 적용하고 확인하며 배울 수 있습니다.

사이트 링크: w3schools.com
